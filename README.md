## 详细：

后缀<code>.conf</code>为配置文件；

<code>.Sgmodule</code>为<code>surge</code>模块文件；

<code>rule文件夹</code>为分流规则；

<code>icon文件夹</code>为图标文件；

<code>protocol文件夹</code>为搭建协议。
